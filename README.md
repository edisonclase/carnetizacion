# Nova ID - Sistema de Carnetización, Control de Acceso y Asistencia  
**by Aula Nova**

---

## 📌 Descripción

Nova ID es una plataforma profesional para centros educativos que integra:

- Identificación digital
- Control de acceso
- Gestión de asistencia
- Facturación institucional

---

## 🎯 Objetivos

- Digitalizar la identidad institucional
- Automatizar accesos y asistencia
- Monetizar el servicio mediante facturación
- Escalar como SaaS multi-centro

---

## 🧱 Arquitectura

- FastAPI
- PostgreSQL
- SQLAlchemy
- Alembic
- JWT
- Passlib + bcrypt

---

## 🔐 Seguridad

- Autenticación JWT
- Roles:
  - super_admin
  - admin_centro
  - registro
  - consulta

- Restricción por centro
- Protección de endpoints

---

## 💰 Facturación

Incluye:

- Facturación por centro
- Registro de pagos
- Balance automático
- Base para analítica financiera

---

## 🏫 Multi-centro

- Separación por `center_id`
- Usuarios restringidos
- Escalable a múltiples instituciones
